PyDom
====

PyDom is a python script for checking domain availability. 

To enable output coloring on OSX, you need to have termcolor package - pip
install termcolor <br>


### Usage
```shell
$ python pydom.py <domain>
```
    
### Example usage
```shell
$ python pydom.py geolymp
```
